## Getting Started

## First-Time Setup

### Step 1: Position Your Camera

**Best Practices:**
- Mount camera above the entry/exit point
- Ensure clear, unobstructed view
- Avoid backlighting (bright windows behind people)
- Height: 7-10 feet is ideal
- Angle: Slight downward angle works best

**Pro Tips:**
- Test during busy hours
- Ensure good lighting
- Minimize blind spots
- Keep lens clean

### Step 2: Draw the Counting Line

1. **Start the Preview**: Camera feed appears automatically
2. **Position the Line**: 
   - Drag the orange circles at each end of the line
   - Position line where people cross when entering/exiting
   - Make sure line is perpendicular to traffic flow

3. **Lock the Line**: Click "ğŸ”“ UNLOCK" button to toggle to "ğŸ”’ LOCKED"
   - Always lock before starting counting!

### Step 3: Set the Direction

The line shows:
- **IN** (Green) - Side people come from when entering
- **OUT** (Red) - Side people go to when entering

If backwards, click "â‡„ FLIP" to reverse.

### Step 4: Start Counting!

Click "â–¶ START" to begin counting. That's it!

## Troubleshooting

### Camera Issues

**Problem**: Camera not showing
**Solutions**:
1. Check camera connection
2. Try different camera number (0, 1, or 2)
3. Grant camera permissions:
   - **Mac**: System Settings â†’ Privacy â†’ Camera
   - **Windows**: Settings â†’ Privacy â†’ Camera
4. Close other apps using camera
5. Restart the application

**Problem**: Black screen
**Solutions**:
1. Ensure adequate lighting
2. Check camera lens is not covered
3. Try different camera

### Counting Issues

**Problem**: Not counting people
**Solutions**:
1. Ensure line is locked (ğŸ”’ LOCKED button)
2. Check if people fully cross the line
3. Verify counting direction (IN/OUT labels)
4. Adjust line position
5. Ensure good camera angle and lighting

**Problem**: Counting incorrectly
**Solutions**:
1. Improve lighting conditions
2. Reposition camera for better angle
3. Ensure people cross line completely
4. Avoid crowded overlapping scenarios
5. Clean camera lens

**Problem**: Double counting
**Solutions**:
- System has built-in 2-second cooldown
- If issue persists, reposition line further from entry
- Ensure people don't linger near line

### Performance Issues

**Problem**: App running slowly
**Solutions**:
1. Close other resource-intensive apps
2. Lower camera resolution if possible
3. Ensure meeting system requirements
4. Restart application
5. Restart computer

**Problem**: High CPU usage
- Normal during counting (AI processing)
- Reduce to ~30-40% when stopped
- Consider more powerful hardware for multiple cameras

### Data & Analytics Issues

**Problem**: Analytics not showing
**Solutions**:
1. Ensure you've counted customers first
2. Wait a few minutes for data to process
3. Check database file exists
4. Restart application

**Problem**: Export fails
**Solutions**:
1. Ensure you have write permissions
2. Choose different save location
3. Check disk space

## FAQ

### General Questions

**Q: Does this work with any camera?**
A: Yes! Any USB webcam or built-in camera works. 1080p cameras provide best accuracy.

**Q: Do I need internet connection?**
A: No! The app works 100% offline. All processing is done locally.

**Q: How accurate is the counting?**
A: 95-98% accuracy in optimal conditions (good lighting, clear view, proper setup).

**Q: Can it count multiple entries simultaneously?**
A: Yes! The AI tracks multiple people at once.

**Q: How much data does it store?**
A: Database file grows ~1 MB per 10,000 customers. Very minimal storage.

### Setup Questions

**Q: What's the best camera position?**
A: Above the entry point, 7-10 feet high, slight downward angle.

**Q: Can I use multiple cameras?**
A: Current version supports one camera. Contact us for multi-camera needs.

**Q: How do I backup my data?**
A: Copy the `customer_analytics.db` file to safe location.

### Feature Questions

**Q: Can it identify specific customers?**
A: It tracks customers by session, but doesn't identify individuals (privacy-focused).

**Q: Can it integrate with my POS system?**
A: Export CSV data can be imported into most systems. Custom integrations available.

**Q: Can I get real-time alerts?**
A: Yes! Configure alerts in the ğŸ”” ALERTS section.

### Licensing Questions

**Q: How many computers can I install on?**

**Q: Can I move my license to a new computer?**

**Q: Is there a free trial?**
