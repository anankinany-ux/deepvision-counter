## Recording Tips

### Equipment Needed
- Screen recording software (OBS, Camtasia, ScreenFlow)
- Quality microphone
- Video editing software
- Consistent lighting for live shots

### Best Practices
1. **Script adherence**: Follow script but sound natural
2. **Pacing**: Not too fast, not too slow - conversational
3. **Visuals first**: Show, then explain
4. **Highlights**: Use arrows, circles, zoom for emphasis
5. **Music**: Subtle background music, not overpowering
6. **Branding**: Consistent intro/outro
7. **Captions**: Add subtitles for accessibility
8. **Length**: Keep videos under 6 minutes when possible

### Post-Production Checklist
- [ ] Color correction
- [ ] Audio leveling
- [ ] Remove "umms" and pauses
- [ ] Add background music
- [ ] Create thumbnails
- [ ] Export in 1080p minimum
- [ ] Upload to YouTube/Vimeo
- [ ] Add timestamps in description
- [ ] Create playlist
